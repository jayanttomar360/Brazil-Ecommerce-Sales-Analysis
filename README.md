# Brazil E-Commerce Sales Analysis

## Project Overview
This is an end-to-end Data Analytics project built using Excel, SQL, PostgreSQL, and Power BI on a Brazil E-Commerce dataset.

The project focuses on sales analysis, KPI tracking, payment analysis, delivery analysis, seller performance analysis, and business insights generation through interactive dashboards and SQL-based data analysis.

- Excel
- SQL
- PostgreSQL
- Power BI
- Data Visualization
- KPI Analysis
- Dashboard Design

## Key Analysis Performed
- Monthly Revenue Trend Analysis
- Revenue by State Analysis
- Payment Distribution Analysis
- Order Status Analysis
- Top Seller Performance Analysis
- Delivery Performance Analysis
